DjYayo - Application Android

Présentation

Ce projet a pour but de proposer une version mobile pour les systèmes Android de la Web Application Dj Yayo.

Le projet ne doit recréer que l’expérience d’utilisation de Dj Yayo. Les ressources utilisées sont les mêmes que celles utilisées par la Web Application.

Pour ce faire, le webservice proposé par Dj Yayo sera utilisé avec l’application Android.

Dj Yayo

Dj Yayo est une web application permettant aux utilisateurs de voter pour la musique qu’ils souhaitent écouter à partir du lecteur Spotify Dj.

Spotify Dj

Les sources de Spotify Dj sont disponibles publiquement sur Github.

SpotifyDj is a new way to share music with friend or with co-workers.
Let everyone a chance to put some music on the speakers.
I plug it in in the breakroom of the school I work for and now you can take a break in music !

  1. Plug a computer on the speaker and launch spotifyDj desktop application
  2. Tell everyone to go on dj.yayo.fr
  3. Everyone can search for music using the entire spotify library and vote for their favorite songs
  4. Music that get the higest number of votes will play on the speakers

Fonctionnalités

Le projet d’application android pour Dj Yayo veut proposer au minimum les mêmes fonctionnalités que celles proposées par la Web Application du même nom. Celles-ci sont détaillées dans les 2 sections ci-dessous.

Espace Utilisateur

  • Espace public
    • Connexion possible via Google+ et Facebook
  • Espace connecté
    • Fonctionnalité de déconnexion
    • Création de room de lecture
    • Jonction à une room existante
  • Espace connecté et joint à une room
    • Fonctionnalité de départ d’une room
    • Interface de visualisation de la file de lecture de la room jointe
    • Interface de recherche de musiques ajoutable à la file de lecture
    • Interface de visualitation de l’historique des musiques jouées
    • Intégration du système de votes

Espace Administrateur

  • Administration des musiques
    • Possibilité de passer à la musique suivante
  • Administration des utilisateurs
    • Possibilité d’ajouter un administrateur
    • Possibilité d’enlever un administrateur
    • Visualisation des utilisateurs s’étant connectés à DjYayo

Evolutions Possibles

  • Intégration d’un système de notifications

  • Intégration d’une barre de notification permanente et/ou d’un widget (SpotifyLike)

  • Intégration du client musical dans le client utilisateur
